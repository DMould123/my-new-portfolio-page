Om Mig

Hej! Jag är David, nyligen utexaminerad med en stark inriktning på samarbete och social interaktion som Fullstack Utvecklare. Med två praktikperioder i bagaget är jag nu ivrig att ta mig an spännande nya projekt. Under mina studier har jag förädlat mina färdigheter inom en rad programmeringsspråk, inklusive JavaScript, HTML, CSS, React, React Native och Angular. Dessutom har jag fått värdefull erfarenhet av databastekniker som MySQL, MongoDB och Firebase. På senare tid har jag också börjat utforska Java.

Min Portfölj

Här är några projekt jag har arbetat med:

HTML, CSS, & JavaScript

Game of Thrones Slumpmässig Citatgenerator

Denna webbapp, Game of Thrones Quote Generator, tillhandahåller slumpmässiga citat från dina älskade Game of Thrones-karaktärer. Den är byggd med HTML, CSS och JavaScript och har FontAwesome-ikoner för en visuellt tilltalande upplevelse.

React

Musik & Filmer

Filmer & Musik är en samling av mina favoritfilmer och album. Detta projekt representerar min resa inom fullstack webbutveckling, med fokus på React för frontend och en robust backend som drivs av Node.js, Express.js och MongoDB.

TypeScript

React Typescript-quizapp

Ett React och Typescript-baserat skolprojekt drivet av Trivia API, erbjuder interaktiva quiz. Användare kan välja bland olika kategorier och svårighetsgrader, svara på slumpmässiga frågor inom en 30-sekunders tidsgräns. Känn dig fri att prova det!

Vue

Punk Beer API Generator

Detta skolprojekt är en enkel webbapp som genererar slumpmässiga öl med Punk API. Den är byggd med Vue och inkluderar en navigeringsfält med länkar till hemsidan, ölgeneratorn och företagsprofil-/kontaktsida.
